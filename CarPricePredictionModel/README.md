## Araba Fiyat Tahmin Modeli

Bu proje, ikinci el arabaların fiyatlarını tahmin etmek için makine öğrenmesi tekniklerini kullanır. Python programlama dili ve popüler veri bilimi kütüphaneleri ile geliştirilmiştir.

### Kullanılan Kütüphaneler
- numpy
- pandas
- matplotlib
- seaborn
- scikit-learn

### Veri Seti
Projede `CarPrice.csv` adlı veri seti kullanılmıştır. Bu veri seti, çeşitli araba özelliklerini ve fiyatlarını içermektedir.

### Proje Adımları
1. **Veri Yükleme ve İnceleme:** Veri seti pandas ile yüklenir ve temel istatistiksel bilgiler incelenir.
2. **Veri Temizleme:** Eksik veriler ve aykırı değerler kontrol edilir.
3. **Veri Görselleştirme:** Fiyat dağılımı ve değişkenler arası korelasyonlar görselleştirilir.
4. **Modelleme:** Karar ağacı regresyon modeli ile fiyat tahmini yapılır.
5. **Model Değerlendirme:** Modelin başarısı çeşitli metriklerle ölçülür.

### Nasıl Çalıştırılır?
1. Gerekli kütüphaneleri yükleyin:
   ```bash
   pip install -r requirements.txt
   ```
2. `app.ipynb` dosyasını Jupyter Notebook ile açın ve adım adım çalıştırın.
